# SourceCodeDownloader
>一个下载器，从一些软件的官方网站下载源代码

###使用方法：
将软件版本号按行写入以软件名命名的txt文件中，并将txt文件存放在versions文件夹下;

使用命令: python downloader.py yoursoftwareName
